# 钢结构旋转楼梯重量计算工具

一个用于估算钢结构旋转楼梯重量的单页网页工具。直接在浏览器中打开即可使用，也可以通过 GitHub Pages 发布成在线网页。

## 功能

- 输入旋转楼梯尺寸与构件参数
- 估算钢结构、踏步、扶手等相关重量
- 单文件 HTML，无需后端服务
- 适合报价前的快速测算和方案预估

## 在线使用

如果启用 GitHub Pages，访问地址通常是：

```text
https://你的GitHub用户名.github.io/stair-weight-calculator/
```

## 本地使用

下载项目后，双击打开 `index.html` 即可。
